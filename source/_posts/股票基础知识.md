### 3、股票的涨跌幅
```language
沪深A股：0和60开头的股票每天的涨跌幅是10%（最高涨10%，最低跌10%，即涨停和跌停）
创业板：3开头的股票每天的涨跌幅是20%
新股：第一天涨跌幅44%（68开头的科创板除外）
科创版：68开头的股票，前5个交易日无限制，后面涨跌幅为20%
ST股：亏损股的涨跌幅是5%
```

### 5、交易规则

```language
股票最低买1手
1手 = 100 股
即：按100股的整数倍来买
例如：每股13元，则一手总共需要1300元

当前股票交易费用分为三部分：佣金、印花税、过户费（仅上海股票收取）
1、印花税：1‰（卖的时候才收取，为国家税收，全国统一）
2、过户费：深圳交易所无此项费用，上海交易所收费标准：按成交金额的0.02‰人民币
3、交易佣金：最高收费为3/10000，最低收费为5/10000，各家券商收费不一。
```

### 6、看盘界面

```language
均线：n天的收盘价相加，除以n天，用线连起来
例如：MA5：5天的均价线，5可以修改为其它想看的数值
	
成交量：n天所有的成交量相加，再除以n天，用线连起来
```

### 7、股票趋势

```language
顺势而为，尽量做上涨趋势的股票，行情好时入手

压力位：当股价上涨快要到达上一次上涨趋势的高位时，为压力位，可以考虑卖出股票
支撑位：当股价下跌快要到达上一次下跌趋势的低位时，为支撑位，可以考虑买入股票
```

### 8、突破和跌破

```language
突破：当股价上涨过程中，价格冲过上一次上涨趋势的高位时，称之为突破，上涨力量很大，可以考虑买入股票
跌破：当股票下跌过程中，股价下跌冲过上一次的下跌趋势低位时，称之为跌破，下跌力量很大，可以考虑卖出股票
```

### 9、买卖机制

```language
T+1机制：今天买的股票，必须过完本天以后，到第二天的交易日开始才可以卖出
```

### 10、如何选股

```language
1、基本面选股：根据k线图走势选股
2、热点题材选股：市场行情热点板块选股
3、事件突发选股：根据突发情况选股
```
